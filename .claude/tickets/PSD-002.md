# PSD-002: PSDエクスポート
- **Package**: packages/adapter-psd
- **Depends**: TYPES-001
- **Complexity**: M

## Description
Export internal Document via ag-psd writePsd. RGB 8bit only. Generate canvas/imageData for each layer. Generate composite image. Export options (text preserve/rasterize).

## Acceptance Criteria
- Exported PSD opens in Photoshop
- Round-trip (import->export->re-import) produces equivalent result
