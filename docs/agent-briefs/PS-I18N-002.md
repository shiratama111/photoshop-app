# PS-I18N-002: 全UI日本語化 仕上げ

## 目的
`PS-I18N-001` で導入した基盤を使い、レンダラ主要UIの英語表示を日本語に統一する。

## 前提
- `PS-I18N-001` マージ済み
- `PS-HISTORY-001`, `PS-TEXT-001`, `PS-TEXT-002` マージ済み（競合回避）

## 編集可能ファイル
- `packages/app/src/renderer/App.tsx`
- `packages/app/src/renderer/components/**`
- `packages/app/src/renderer/store.ts`（ユーザー表示メッセージのみ）
- `packages/app/src/renderer/i18n/messages.ts`

## 実装要件
1. 主要UI文字列をキー化し、日本語辞書へ移動。
   - Toolbar
   - Sidebarタブ
   - History
   - Text Properties
   - Layer Style Dialog
   - Status bar
2. ステータスメッセージを日本語化。
3. 新規追加UIは英語ベタ書き禁止。

## 受け入れ条件
- 通常操作で見える主要文字列が日本語。
- 表示崩れ・文字化けなし。
- 既存ショートカットや機能動作は不変。

## テスト
- 主要画面のスモークテスト
- i18nキー欠落チェック（簡易でも可）
