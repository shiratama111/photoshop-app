# PS-I18N-001: i18n基盤 + メニュー日本語化

## 目的
まずは「メニュー画面が英語」の問題を解消し、今後の全UI日本語化に耐える基盤を入れる。

## 編集可能ファイル
- `packages/app/src/main/menu.ts`
- 新規: `packages/app/src/renderer/i18n/messages.ts`
- 新規: `packages/app/src/renderer/i18n/index.ts`
- 必要最小限の呼び出し側

## 実装要件
1. `main/menu.ts` のメニュー表示を日本語化（Photoshopに近い語彙）。
   - 例: `File` -> `ファイル`, `Edit` -> `編集`, `Image` -> `イメージ`, `Layer` 相当があれば `レイヤー`
2. 将来の多言語化向けに `t(key)` ベースの最小i18n実装を追加。
3. 文字化けしている文言（mojibake）をUTF-8で修正。
4. ショートカットは現行維持。

## 受け入れ条件
- アプリメニューが日本語表示。
- 新規i18nモジュールが追加され、他チケットが再利用可能。
- 文字化け文言が減っている（新規追加分はゼロ）。

## テスト
- `menu.ts` のユニットテスト（ラベルの主要項目を検証）
- i18n関数のキー未存在時フォールバックテスト
