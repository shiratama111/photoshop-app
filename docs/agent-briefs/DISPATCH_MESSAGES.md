# AIエージェント送信用メッセージ（コピペ用）

このファイルは、各AIエージェントにそのまま送る依頼文です。

## Wave1: 同時送信（4本）

### 1) Agent-A 送信文（PS-PAN-001）
```
担当チケット: PS-PAN-001
ブランチ: feat/ps-pan-001-space-pan
指示書: docs/agent-briefs/PS-PAN-001.md

要件を満たす実装とテスト追加まで実施してください。
完了時に以下を必ず報告してください。
1. 変更ファイル一覧
2. 実行したテストコマンドと結果
3. 手動確認手順と結果
4. 既知の未解決事項（あれば）

制約:
- 他チケット領域のファイルは編集しない
- コミットメッセージは `PS-PAN-001: ...` 形式
```

## Integrator向け最終確認テンプレ
```
各チケットの報告を受領後、以下を確認すること。

- lint/testが通っているか
- 変更ファイルが担当範囲内か
- 主要受け入れ条件が満たされるか
- 既知課題の影響範囲
```
